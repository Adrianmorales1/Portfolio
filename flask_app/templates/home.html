MY PORTFOLIO

Test

This blog application allows users to create journal entries about their plants using Java, Bootstrap, and MySQL. It was designed using Balsamiq Wireframe and displayed all HTML files using Bootstrap for a responsive design. It was developed in SpringToolSuite as a source-code editor to improve functionality and used back-end validations to verify user input and maintain data consistency. The application also includes the ability to upload and store images of plants using object-oriented programming, allowing users to display different plant images.

Programmed With:

Java - SpringToolSuite - Spring - Bcrypt - Bootstrap - MySQL - HTML - CSS - Github

Song Review

This application allows users to create and like reviews using Spotify's API, Python, Flask, and MySQL. As the project lead, I led the team through brief and quick conversations about the project wireframe and the ERD in MySQL Workbench and received input as needed as well as delegated roles and responsibilities to 3 other developers on the team. Through those two weeks I provided motivation and opportunities for technical learning. Validations and Bcrypt were instituted for secure login and registration. Some of the big learning opportunities of this project was requesting and displaying song information from Spotify's API as well as getting everyone accustomed to merging code into the main branch by using Git in our group repository on github.

Programmed With:

Python - Flask - MySQL - Jinja- Bcrypt - Bootstrap - Spotify API - HTML - CSS - Git
